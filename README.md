# 📁 College Data Repository

This repository contains structured datasets (CSV and JSON files) provided by my college for academic and practice purposes.

## 📌 Contents

* 📄 **CSV Files** – Tabular datasets for analysis and processing
* 🗂️ **JSON Files** – Structured data for applications and testing

## 🎯 Purpose

The goal of this repository is to:

* Store and organize datasets provided by the college
* Use the data for learning and practicing data analysis
* Serve as a reference for projects and assignments

## 🛠️ Usage

You can use these files for:

* Data analysis (Python, R, Excel, etc.)
* Practice with data visualization tools
* Testing APIs or applications with sample data
